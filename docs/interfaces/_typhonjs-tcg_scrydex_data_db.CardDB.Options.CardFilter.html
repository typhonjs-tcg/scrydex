Interface CardFilter

Provides the configuration object for CardFilter.filter.

interface CardFilter {
    properties: {
        border?: Set<string>;
        cmc?: number;
        colorIdentity?: Set<string>;
        formats?: string[];
        keywords?: RegExp[];
        manaCost?: string;
        price?: PriceFilter;
    };
    regex?: {
        fields: Set<string>;
        log: {
            caseInsensitive: boolean;
            exact: boolean;
            input: string;
            wordBoundary: boolean;
        };
        op: RegExp;
    };
}

Type Declaration

  • Optionalborder?: Set<string>

    Card border colors to filter.

  • Optionalcmc?: number

    Match card CMC.

  • OptionalcolorIdentity?: Set<string>

    WUBRG color identity set.

  • Optionalformats?: string[]

    Game format legality.

  • Optionalkeywords?: RegExp[]

    An array of RegExp instances for keywords that a card uses such as 'Flying' and 'Cumulative upkeep'.

  • OptionalmanaCost?: string

    Match exact mana cost.

  • Optionalprice?: PriceFilter

    Price filter to match.
